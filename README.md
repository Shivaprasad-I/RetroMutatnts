# RetroMutatnts

In this I'll be trying to create classic games which can be run in a browser.

**Q: Don't we already have sites which actually do this?**
- Yes, but those are not built by me.

**Q: AI can build same thing in minutes?**
- Yes, but where's the fun in that?

**Q: So you'll not be using AI at all in this project?**
- No, I'm not crazy and I suck at cosmetics (I'm talking about you CSS) so I rely entirly on AI for that, but logic that'll be only me

### Why am I here?
Honestly? My day job has become a lot of "watching AI do the work," and my brain is starting to feel like mush. I needed a place to silence the "itch" to actually write code again. This is my playground for building classic 2D games with a modern twist. 

Instead of grinding boring "solve this math puzzle" sites (sorry LeetCode, I still love you but you're dry), I’m learning how computers *actually* work by building things. 

### The Master Plan:
- **Classic Games:** Starting with a simple Snake (I’ve already got a JavaScript one running!), then maybe Tetris, or whatever else feels fun.
- **Going Under the Hood:** I'm moving away from the easy stuff to play with "low-level" languages. It’s like learning to fix a car engine instead of just driving an automatic. I want to understand how memory works and where all those ones and zeros actually go.
- **Purposeful Practice:** I’ve used more logic building a simple snake game than I have in 3 years of "real world" experience. This is about making things that *do* something.
